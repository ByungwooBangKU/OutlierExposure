# OutlierExposure

Welcome to the OutlierExposure repository!

This project aims to provide tools and methods for outlier exposure in machine learning models, helping improve robustness and generalization by leveraging out-of-distribution data.

## Features

- Outlier exposure dataset utilities
- Model training scripts
- Evaluation tools

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/ByungwooBangKU/OutlierExposure.git
   ```

2. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Explore the code and documentation for usage examples.
